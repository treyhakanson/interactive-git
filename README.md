# Interactive Git

A simple CLI to run checkouts and branch deletions interactively.

Installation:

```
npm install -g interactive-git
```

Using the CLI:

```
gi [command]

Commands:
  gi c  checkout a branch
  gi d  delete a branch

Options:
  --version  Show version number                                       [boolean]
  --help     Show help                                                 [boolean]
```
